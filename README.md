# Blog Content Structure

This is your organized folder structure for better content management:

## Directory Structure

```
content/
├── personal/           # Personal updates, reflections, general thoughts
│   ├── welcome.md     
│   └── second.md      
├── technical/         # Technical deep-dives, tutorials, research
│   └── (future technical posts)
├── pages/            # Static pages (About, Contact, etc.)
│   └── about.md      
├── images/           # Image assets for blog posts
└── videos/           # Video assets for blog posts
```

## Content Guidelines

### Personal Posts (`personal/`)
- Life updates
- Career reflections  
- Learning journeys
- General thoughts

### Technical Posts (`technical/`)
- Deep technical tutorials
- Research explanations
- Code walkthroughs
- Project showcases

### Pages (`pages/`)
- About page
- Contact information
- Static content

### Assets (`images/`, `videos/`)
- Store all media files here
- Reference with relative paths: `../images/filename.jpg`
